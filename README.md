## How to Run the Study

To replicate the findings of this research, please follow the steps outlined below:

1.  **Install Dependencies**: Before running the scripts, ensure that you have all the necessary libraries installed. You can do this by running the following command in your terminal:

    ```
    pip install -r requirements.txt
    ```

2.  **Data Acquisition**: The first step is to gather all the necessary data. This is handled by the `data_btc.py` script, which collects information from various sources such as Yahoo Finance, Google Trends, and Wikipedia. To run it, simply execute the following command in your terminal:

    ```
    python data_btc.py
    ```

    Upon successful completion, a file named `btc_macro_dataset.csv` will be generated in your project's root directory.

3.  **Descriptive Statistics and Plots**: To gain a better understanding of the collected data, you can generate descriptive statistics and relevant plots. This is done by running the `descriptive_stats.py` script:

    ```
    python descriptive_stats.py
    ```

4.  **Main Analysis**: The core of the study is performed by the `run_study_btc.py` script. This script conducts the primary analysis and generates the main results of the research. To execute it, use the following command:

    ```
    python run_study_btc.py
    ```

## File Descriptions

*   **data_btc.py**: This script is responsible for fetching all the required data from various sources.
*   **descriptive_stats.py**: Generates descriptive statistics and visualizations for the collected data.
*   **correlation_heatmap.py**: Creates a correlation heatmap to visualize the relationships between different variables in the dataset.
*   **run_study_btc.py**: This is the main script that carries out the core analysis of the study.
*   **btc_study_visuals/**: This directory will be automatically created to store all the plots and tables generated by the scripts.
